# Grove LCD RGB

- schließe das LCD an einen der I2C pins des Grove-Shields an.
- lade das Beispiel auf den Arduino
- auf dem LCD wird nun ein Text angezeigt.

## Aufgabe

- ändere den Code so ab, das in der ersten Zeile 'Easy' und in der zweiten Zeile 'Sommer' steht.
- und ändere die Hintergrund-Farbe z.B. auf ein schönes Violett ;-)

## Weitere Informationen
